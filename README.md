# JOEMET123 — Official Website

Static website for Joemet123, a Call of Duty Zombies content creator.

## Live Site

Hosted on GitHub Pages at `https://joemet123.com` (or `https://<username>.github.io/joemet123-site/`).

## Structure

```
joemet123-site/
├── index.html              ← Home (Twitch embed, donate, merch, socials, YouTube carousel)
├── about.html              ← About / Mission Statement
├── live.html               ← Live Stream page
├── affiliate-discounts.html ← Affiliate partner codes
├── paid-meatbawl.html      ← Subscribe / Membership perks
├── chat-rules.html         ← Chat rules
├── commands.html           ← Twitch chat commands
├── assets/
│   ├── css/style.css       ← Shared stylesheet
│   ├── js/main.js          ← Shared JavaScript
│   └── images/             ← Placeholder for future images
└── README.md
```

## Tech

- Pure HTML / CSS / JavaScript — no frameworks or build tools
- Dark theme with fiery orange (#FF6B00) and red (#FF2D2D) accents
- Mobile-first responsive design
- Hamburger navigation on mobile

## Hosting on GitHub Pages

1. Push this folder to a GitHub repo named `joemet123-site`
2. Go to **Settings → Pages**
3. Set source to **main** branch, root (`/`)
4. (Optional) Add a custom domain: `joemet123.com`
